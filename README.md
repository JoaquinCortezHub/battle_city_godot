# Battle City Godot

Prototipo académico para Programación Avanzada inspirado en Battle City, desarrollado con Godot 4 y GDScript.

## Estado actual

- Proyecto Godot 4 inicializado.
- Escena principal jugable.
- Tanque del jugador con movimiento en grilla libre y disparo.
- Enemigos con movimiento automático y disparos.
- Bloques de ladrillo destructibles, bloques metálicos, paredes y base a defender.
- HUD con puntaje, vidas, enemigos restantes y estado de partida.

## Controles

- `W`, `A`, `S`, `D`: mover el tanque.
- `Espacio`: disparar.
